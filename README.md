A simple, classic Snake game built with plain HTML, CSS, and JavaScript.

## How to Play

1. Open `index.html` in your web browser
2. Use arrow keys or WASD to control the snake
3. Press SPACE to pause/resume the game
4. Eat the red food to grow and score points
5. Avoid hitting the walls or yourself!

## Features

- Classic snake gameplay
- Score tracking with local high score storage
- Pause/resume functionality
- Responsive keyboard controls (Arrow keys + WASD)
- Clean, minimalist design

## Running the Game

Simply open `index.html` in any modern web browser. No installation or build process required!

## Controls

- **Arrow Keys** or **WASD**: Move the snake
- **SPACE**: Pause/Resume
- **New Game Button**: Start a new game
- **Pause/Resume Button**: Toggle pause

Enjoy the game!
